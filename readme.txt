Wordpress sc-pay-per-post Plugin, more flexible and more secure post plugin.

== Description ==

SC Pay per Post allows only the registered users who can post content through paying some amount and only paying members can view this content. WP Pay Per Post plugin is integrated with pay pal account. This plugin will provide you more security so only paid members can access the post or content. You can easily install the plugin and setup on your WordPress site and it is Compatible up to Wordpress Version: 3.0.0 or higher. So if you need a simpler, secure and more effective WP pay per post plugin then download the plugin now.

Check www.solvercircle.com for more information.

== Installation ==

1. Upload 'sc-pay-per-post' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "SCPPP" menu in your WordPress admin panel.
4. To setup SC Pay Per Post Plugin, first create three page named Post Page, Post Dashboard, and a subscription page. Use the following short code in those three pages to work this plugin (without any space).

Post Page : [scppppostpage]
Post Dashboard : [scpppmanagepost]
Subscription : [scpppuserupgrade]

5. You need an API key to run the SC Pay Per Post Plugin (You will get API key at www.solvercircle.com/contact-us page by submitting your name, domain name and email address. We will provide you the api key by email).
6. Place the API key at SCPPP Upgrade menu.
7. Enjoy with the plugin.
